 16. Problem 5. Maximal area sum
 17. 
 18. Write a program that reads a text file containing a square matrix of numbers.
 19. Find an area of size 2 x 2 in the matrix, with a maximal sum of its elements.
 20. The first line in the input file contains the size of matrix N.
 21. Each of the next N lines contain N numbers separated by space.
 22. The output should be a single number in a separate text file.
 23. Example:
 24. 
 25. input	output
 26. 4 
 27. 2 3 3 4 
 28. 0 2 3 4 
 29. 3 7 1 2 
 30. 4 3 3 2	17
